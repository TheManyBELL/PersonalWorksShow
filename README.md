## 02 [增强现实飞机装配](./ARAssembly)
**项目描述** 
为用户提供飞机装配引导。用户佩戴增强现实头盔，通过悬浮面板自由控制飞机装配进度。装配包括软管和刚体两部分，装配动画由脚本自动批量生成。  
**相关技术** MRTK ; Unity ; Hololens2
